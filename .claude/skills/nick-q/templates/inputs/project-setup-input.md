# Project Setup Input

新規プロジェクト開始時の入力フォーム。
Vision + Screen + Domain を一括で作成し、Feature Drafts を生成するための情報を記入してください。

**入力のポイント:**
- 詳細に書けば書くほど、AI との QA のやり取りが減ります
- 最低限「プロジェクト名」と「一言で説明すると」があれば開始できます
- 機能リストは「機能 1」から始めて、必要に応じて追加してください
